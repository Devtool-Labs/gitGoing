## Usage

Use gitGoing while collaborating with colleagues, especially while troubleshooting.  GitGoing is also perfect for introducing people to git and its functionality.  

<hr>
## Requirements

- React 2.6.x
- Express.js 4.13.x
- Node 5.8.x
- Redis 2.6.x
